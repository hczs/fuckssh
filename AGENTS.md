# fuckssh

跨平台 CLI：围绕标准 `~/.ssh/config` 提供 VPS 一站式 SSH 配置、列表与搜索。


## 注意
- 此仓库是面向初学者的仓库，代码要干净易懂，同时要保证可维护性
- 代码要符合 go 语言最佳实践，可供初学者学习
- 代码要适当的加中文注释，为初学者讲解
